# Aegis

> **WARNING: This is still a work in progress.**

Simple and secure user authentication and authorization for your applications.

## What is Aegis?

Aegis is a lightweight authentication RESTful API designed to provide secure user authentication and authorization for applications.

## Features

- User registration and login
- Password hashing and security
- Token-based authentication (JWT)
- Role-based access control

## Why use Aegis instead of building your own or using other solutions?

No reason. Aegis is a personal project to learn Go while building something useful. If you find it useful, that's great! If not, there are many other established solutions available.
